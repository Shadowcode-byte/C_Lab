# C_Lab
This repository contains all the C programming lab assignments, practical questions, and practice programs completed as part of the C Programming Lab.

Programs are organized topic-wise for easy understanding:
Basic Programs (I/O, operators, conditions)
Looping Constructs (for, while, do-while)
Functions & Recursion
Arrays & Strings
Pointers
Structures & Unions
File Handling
Dynamic Memory Allocation

Objectives
To develop strong fundamentals in C
To apply programming logic in solving problems
To learn arrays, functions, pointers, structures & file handling
To write efficient and modular C programs

Suggestions and improvements are welcome through Issues or Pull Requests.
